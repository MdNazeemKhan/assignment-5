What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each node corresponds to part of the document (like elements, attributes, or text). JavaScript can interact with the DOM to dynamically update, delete, or add content, allowing for real-time changes to a web page.

How do you select an element from the DOM?

1. getElementById(): Selects an element by its ID.

2. getElementsByClassName(): Selects elements by class name.

3. getElementsByTagName(): Selects elements by tag name.

4. querySelector(): Selects the first element that matches a CSS selector.

5. querySelectorAll(): Selects all elements that match a CSS selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element instead of individual child elements. This works because events bubble up from the target element to its ancestors, allowing the parent to handle events triggered by its children.


Why it's useful:


1. Efficiency: Instead of adding multiple listeners to many elements, a single listener on a parent can handle all events from child elements, reducing memory usage.
